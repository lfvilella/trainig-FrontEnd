About Gym Project

O “Gym Project” é um projeto para testar as skils de desenvolvimento web. Onde simula uma situação real, simples no caso deste sistema, mas com o intuito de apenas treinar/praticar e aprender conteudo.
O sistema simula uma aplicação real de uma academia, onde nele pode efetuar cadastro de alunos nas diversas modalidades que a academia proporciona. Por fim, o sistema automatiza a cobrança mensal dos alunos, sendo assim todo dia primeiro do cada mês, se o aluno não tiver pago ele gera uma cobrança. E para facilitar o administrador, na aba Pagamentos tem o campo 'phone_number' onde só basta clicar nele que é redirecionado para o whatsapp na conversa do aluno.
